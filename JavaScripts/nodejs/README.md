# 安装nodejs
[清华大学开源软件镜像网站](https://mirrors.tuna.tsinghua.edu.cn/)  

下载源码压缩包并解压。  

```sh
$ ./configure
$ make
$ sudo make install
```

安装n：  
```sh
$ sudo npm install n -g
```
