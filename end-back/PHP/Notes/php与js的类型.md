# 一、 数据类型
* JavaScript：  
  Undefined（未定义）, Null（空值）, Boolean(布尔型), Number（数字）, String（字符串）, Object（对象）  
* PHP:  
  NULL（空值）, Boolean（布尔型）, Integer（整型）, Float（浮点型）, String（字符串）, Array（数组）, Object（对象）
  
# 二、 `typeof` 和 `var_dump()`

| 值 | JavaScript(typeof) | PHP(var_dump())
| --- | --- | --- |
| 未定义的值 | 'undefined' | 'NULL' |
| 布尔值 | 'boolean' | 'bool' |
| 整数 | 'number' | 'int' |
| 浮点数 | 'number' | 'float' |
| 字符串 | 'string' | 'string' |
| null | 'object' | 'NULL' |
| 函数 | 'function' | / |
