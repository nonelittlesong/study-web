# 安装
创建工作目录：  
```
$ mkdir myapp
$ cd myapp
```
创建 package.json 文件：  
```
$ npm init
```
这里需要输入几个参数，大部分可以默认。需要注意的是：  
```
entry point: (index.js)
```
可以自定义程序入口，如`app.js`。

安装 `express`：  
```
$ npm install express
```
