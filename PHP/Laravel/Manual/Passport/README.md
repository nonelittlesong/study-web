# 一、 安装
```
composer require laravel/passport
```
>注：如果安装过程中提示需要更高版本的 `Laravel：laravel/passport v5.0.0 requires illuminate/http ~5.6`，可以通过指定版本来安装 `composer require laravel/passport=~4.0`。
