有三种方式来添加字体链接：  
* Standard方式：  
  ```
  <link rel='stylesheet' type='text/css' href='http://fonts.googleapis.com/css?family=Condiment'>
  ```
* @import方式：  
  ```
  @import url(http://fonts.googleapis.com/css?family=Condiment);
  ```
